Ran this code in June 2025, after a year of inactivity. The GNBF_3_retrieve code takes about a half hour to run. The linear algebra routines of scipy uses in GNBF_3_retrieve.ipynb did not seem to be compatible with Anaconda-installed scipy. So, in Documents/Repositories/ice2021/Installing_scipy.ipynb, I forced installation of scipy version 1.13.0. Then GNBF_3_retrieve.ipynb worked fine.
